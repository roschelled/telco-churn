# telco-churn
# Instructions
1. Download the datset from Kaggle : https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset
2. Download the Google Colab
3. Upload the Google Colab file as a new notebook to Google Colab
4. Ensure that the dataset file is uploaded to it, if not upload it.
5. Click Run All in Google Colab
6. Feel free to test the ML predictive model by tampering with the sample_customer data values in the last cell

Requirements: 
pandas
matplotlib
scikit-learn
